# ☀️ Weather App (Java Full Stack)

A simple weather application built with **Spring Boot** (Backend) and **HTML/CSS/JS** (Frontend).

## 🚀 How to Run
1. Clone the repo.
2. Add your OpenWeatherMap API key in `application.properties`.
3. Run `mvn spring-boot:run`.
4. Open `http://localhost:8080`.

## 🛠️ Tech Stack
- Java 17
- Spring Boot 3.2
- Maven
- REST API
- Vanilla JS
